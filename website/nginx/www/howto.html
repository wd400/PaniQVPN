How to use?

Before buying a license please verify that you can successfully set the required setup.

  • Install the openvpn client supporting post-quantum cryptographic primitives:
    Linux 8a034a014b30e8b99044d8e75c19f8ae63b982682721628d2dfe68cffa4c1b6924db761341f17dabbeb6c475b0c659f89e8b8535384d3115bea0cb3a476aeafd (SHA3-512)
    (extract with sudo tar -xvzf pq-openvpn-linux-staged.tar.gz -C /)
    Windows 4b1ff7e7c47d3a8b0c2dffaa80716b22fa7d6025b19054ac224a91edaded854f769c0299082e53df3309fbb73e5097a116b3e0b1f8e197be98a43f5bccd89f35 (SHA3-512)
    You can get them directly from the github releases or (the best option) compile the binary from the sources also provided on the link.

  • Get the configuration file for openvpn, save it on your computer
    Linux Windows

  • Install Tor on your computer (if you're here, chances are you already have).
    Launch the tor browser (or just tor on linux by launching the tor command) which you can then put in the background.

  • Start openvpn with the configuration file
    openvpn --config yourconfigfile.config
    Enter your previously obtained credentials (if you don't have them yet you can get them here )
    Done! You can check that the connection is made with post-quantum algorithms in the openvpn logs (during the key exchange) You can also check that you are using tor by going to this site https://check.torproject.org (clearnet) and that you can access hidden services https://3g2upl4pq6kufc4m.onion/ (DuckDuckGo) .

tested on ubuntu 18.04
